# 關於

楊峻宇老師，師從張大千傳人施雲翔、李苦禪弟子陳琦及李顯亮等，得清華美院橋森教授、國畫大師谷寶玉先生（李苦禪弟子）鼓勵指導，工花鳥和潑彩山水，作品被榮寶齋收藏。

現爲國夢山海書畫院院長，中國書畫篆刻家協會會員，大風堂張大千研究會會員，盛世藝術館首席畫家，青島農業大學客座教授、碩陽考研基地客座教授。
