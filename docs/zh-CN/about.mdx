# 关于

<img src="/assets/member/junyu-yang.png" align="right" width="200" />

杨峻宇老师，师从张大千传人施云翔、李苦禅弟子陈琦及李显亮等，得清华美院桥森教授、国画大师谷宝玉先生（李苦禅弟子）鼓励指导，工花鸟和泼彩山水，作品被荣宝斋收藏。

现为国梦山海书画院院长，中国书画篆刻家协会会员，大风堂张大千研究会会员，盛世艺术馆首席画家，青岛农业大学客座教授、硕阳考研基地客座教授。
